# Phenome-wide analysis of downstream health outcomes following second-line antidiabetic agent prescriptions in All of Us

This repository contains R scripts used to carry out analyses as described a the above titled manuscript. The manuscript is currently provisionally accepted. Journal information will be added once acceptance is finalized.

The version of the repository at the time of finalizing the manuscript received a DOI via Zenodo (`v1.0.1`: [![DOI](https://zenodo.org/badge/995626098.svg)](https://doi.org/10.5281/zenodo.19190669)).
